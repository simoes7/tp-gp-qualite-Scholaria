## 📖 Description
Ce projet a pour objectif d’appliquer les bonnes pratiques de gestion du code, de versionning et de qualité logicielle en utilisant :
- Git & GitHub  
- GitFlow  
- Gestion des issues  
- Travail avec branches  
- Revues de code (code review)

Le projet **Scholaria** simule une application permettant de gérer efficacement un système (à adapter selon votre vrai projet si besoin).

---

## 🛠️ Technologies utilisées
- PHP  
- MySQL  
- HTML / CSS / JavaScript  
- Git & GitHub  

---

## ⚙️ Méthodologie
Nous avons travaillé en utilisant la méthode **Scrum** :
- Organisation en sprints  
- Répartition des tâches via GitHub Issues  
- Collaboration en équipe avec GitFlow  
- Suivi de l’avancement du projet
